View Name:A.ENTITYAUDIT

Description: Provides simple access to the entity change audit log information when the entity "create amend log" options are switched on


Field List:

Field NameData TypeDescription
EntityTypecharacter(15868)Entity type updated - "CO" for contact, "OR" for organisation, "CL" for client, "MA" for matter
NewEntityRefcharacter(15868)New entity reference - contact ID, organisation ID, client code or matter code
OldEntityRefcharacter(15868)Original entity reference - contact ID, organisation ID, client code or matter code
UserIdcharacter(15868)The user ID who made the update
AuditDateTimedatetimeThe date and time of the change
ActionfixcharThe type of action captured by the audit log - either 'Created', 'Updated' or 'Deleted'

* Exactly one of these three flags will always be set to TRUE.
