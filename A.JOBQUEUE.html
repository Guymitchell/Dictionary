View Name:A.JOBQUEUE

Description: Provides a simplified view of the "queued jobs" tables (PUB.QUEUE-REQUEST and PUB.QUEUE-DATA), showing all of the background process jobs that are in the queue, including completed jobs, queued jobs and failed jobs. The job queue is used for EFT remittance generation and will be used for other areas in future.


Field List:

Field NameData TypeDescription
QueueEntryIdintegerUnique job number.
QueueItemTypeIdcharacter(254)Value indicating the type of job for each entry. Each type of background job has a different queue item type identification code
QueueItemTypefixcharShort user-friendly description of the type of job.
ScheduledDateTimedatetimeDate & time the job entry was created on the job queue.
Completedlogical (true/false)TRUE if the job has been processed, FALSE if the job is queued waiting for processing.
CompletedDateTimedatetime[Awaiting description]
Successlogical (true/false)TRUE if the job was completed successfully with no errors, FALSE if the job was run but failed.
ResultMessagecharacter(254)The message returned on completion of the job. If the job failed this will contain any error messages.
DataSeqIdintegerEach queue entry has one or more "chunks" of data associated with it. This field tracks the data sequence number for each queue entry, i.e. this will be 1 for the first set of data associated with each job, 2 of there is more associated data than can fit into 1 data chunk, etc.
DataFragmentcharacter(31000)The chunk of data associated with the queue entry and DataSeqId
