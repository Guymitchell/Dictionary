Table Name:SOSANALY.ANALY-BILL-HIST

Description: If installed, this view will retrieve data from the SOS data warehouse, returning bill life-cycle history including payment values and oustanding values for all bills for previous years and periods


Field List:

Field NameData TypeDescription
BILL-DATEdateBill date
BILL-NOcharacter(32)Bill number
BRANCHcharacter(6)Branch code
CL-CODEcharacter(30)Client code (of matter of bill)
GROSS-VALUEdecimalTotal original bill gross value (i.e. including VAT)
LATEST-PAYMENT-DATEdateLatest payment date the period/year defined in OUTST-PERIOD / OUTST-YEAR
LEAD-FE-DEPTcharacter(8)Matter fee earner department code
LEAD-FEE-EARNERcharacter(8)Matter fee earner code
MT-CODEcharacter(40)Matter code
MT-DEPTcharacter(8)Matter type department code
MT-TYPEcharacter(8)Matter type code
OUTST-PERIODintegerFinancial period in which bill was still outstanding
OUTST-VALUEdecimalOutstanding value of the bill at the end of the year and period defined in OUTST-PERIOD / OUTST-YEAR
OUTST-YEARcharacter(8)Financial year in which bill was still outstanding
PAID-VALUEdecimalAmount paid for this bill in this period and year (OUTST-PERIOD / OUTST-YEAR)
POST-PERIODintegerFinancial period in which bill originally posted
POST-YEARcharacter(8)Financial year in which bill originally posted
SUPERVISORcharacter(8)Matter supervisor code
TRANS-NOintegerBill transaction number (effectively NOMTRANS.TRANS-NO in main SOS database)

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
