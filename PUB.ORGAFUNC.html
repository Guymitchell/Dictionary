User Interface Notes

This table holds the links between organisations and their marketing functions, as set up in the Practice Manager organisation relationships tab. Existing links are shown in the "Function" panel at the top-right of the relationships tab. Functions can be added or removed by right-clicking on this panel.




    


Note that there can be multiple records containing the same ORG-REFERENCE, as there can be multiple function codes assigned to one organisation. It is not compulsory to assign a function to an Organisation so records may not be present for every organisation and care must be taken if linking to the ORGANISATION table.


Linking And Application

A list of organisations sorted and grouped by their function, or a list of Organisations not assigned a function:

ORGANISATION.ORG-REFERENCE -> ORGAFUNC.ORG-REFERENCE (with outer-join)
ORGAFUNC.FUN-CODE -> FUNCTIONS.FUN-CODE (to provide the function description)
