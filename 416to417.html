Data Dictionary Changes Between V4.16 and Vx.17

Tables Removed From Vx.17

The following V4.16 database tables have been removed in Vx.17:

  • CLIEADDDATA (Custom data for clients - all custom data for all entity types has been stored in ADDDATA since PM V4.16 0608 Patch 8, August 2009)
  • CONTADDDATA (Custom data for contacts - now in ADDDATA as above)
  • MATTADDDATA (Custom data for matters/archives - now in ADDDATA as above)
  • ORGAADDDATA (Custom data for organisations - now in ADDDATA as above)
  • CLIEADDGROUPS (Links between clients and custom tab groups - now stored in CUSTOM-LINK)
  • CONTADDGROUPS (Links between contacts and custom tab groups - now stored in CUSTOM-LINK)
  • MATTADDGROUPS (Links between matters and custom tab groups - now stored in CUSTOM-LINK)
  • ORGAADDGROUPS (Links between organisations and custom tab groups - now stored in CUSTOM-LINK)
  • GENERALACTIONS (table never used)
  • PAPSETTINGS (table never used)
  • PARAMS (table never used)

New Tables Added In Vx.17 - Billing

The following database tables have been added specifically for the new billing system:

  • BILL-ADDITIONAL - Additional data for new-style bills, especially Legal Aid CDS7 details
  • BILL-ANTITIME - Anticipated time included in new-style bills
  • BILL-BATCH - Used to link a number of bills together as part of a single run e.g. interim or client billing process
  • BILL-BATCH-OPTIONS - Used in conjunction with BILL-BATCH to preselect options used for e.g. client billing process
  • BILL-COSTALLOC - Fee earner cost allocations in new-style bills
  • BILL-DATA-SAVED - Holds the categories of data have been saved during the processing of a new-style bill (costs, disbursements, anticipated time etc.)
  • BILL-DISB - Details of disbursements included in new-style bills
  • BILL-FIXED-FEE - A copy of the fees record(s) that were selected at the point of creating a Legal Aid Civil/Criminal fixed fee bill
  • BILL-HEADER - Bill header records, effectively the equivalent of BILLDOC for new-style bills created in Vx.17
  • BILL-LA - Additional Legal Aid details for new-style bills
  • BILL-MEMO-LINK - Links between a new-style memo bill and the previous new-style memo bills that the new bill includes
  • BILL-MEMO-PAYMENT - Payments made against new-syle memo bills
  • BILL-MEMO-RECOUPMENT - List of previous memo payments being recouped in current bill
  • BILL-RECIPIENT - List of recipients for either split net/VAT bills or multi-party bills
  • BILL-TIME - Details of time included in new-style bills
