View Name:A.CLIGROUPALL

Description: Returns one row for every client indicating the number of client groups the client relates to, and either the client code / client name for ungrouped clients or client group code / group description for grouped clients. This view is designed to be used in combination with other views to provide client group summary results for grouped clients combined with client level results for ungrouped clients. This view is guaranteed to only ever return one row per client so will not duplicate or inflate figures when joined to other tables where clients are in multiple groups.


Field List:

Field NameData TypeDescription
CL-CODEcharacter(254)Client code
NUMGROUPSint64Number of client groups that the client belongs to
GROUP-OR-CLI-CODEcharacter(255)If the client is not in any group, this column will return the client code. If the client is in exactly one group, this column will return the client group code. If the client is in more than one group this column returns "(x groups)".
GROUP-OR-CLI-NAMEcharacter(475)If the client is not in any group, this column will return the client full derived name (PUB.CLIDB.FULL-DERIVED-NAME). If the client is in exactly one group, this column will return the client group description (PUB.GROUPS.DESCRIPTION). If the client is in more than one group this column returns the text "(Multiple groups)".
