View Name:A.CLIGROUPCOUNT

Description: Summarises the number of clients allocated into each client group. Clients not any group are not included.


Field List:

Field NameData TypeDescription
CL-CODEcharacter(254)Client code
FULL-DERIVED-NAMEcharacter(474)Client name (taken from PUB.CLIDB.FULL-DERIVED-NAME)
NUMGROUPSint64Number of client groups to which the client belongs, which is always 1 or more as clients not in any group are excluded
