Table Name:SOSANALY.ANALY-CALENDAR

Description: If installed, this view will retrieve data from the SOS data warehouse, returning historical financial accounting calendar information derived from the current financial calendar. Each year, quarter and period entry in this calendar appears as a separate row.


Field List:

Field NameData TypeDescription
DAYS-IN-PERIODdecimalThe exact number of elapsed days in this year, quarter or period
PERIOD-ENDdatePeriod end date, i.e. the last day of the year, quarter or period
PERIOD-LEVELcharacter(2)Indicates period type, "Y" for year, "Q" for quarter, or "P" for period. Note that quarterly warehouse summary data is only available on client systems running a 12 period year.
PERIOD-QTRintegerThe period number (1-13) or quarter number (1-4). For year entries, this is always 1.
PERIOD-STARTdatePeriod start date, i.e. the first day of the year, quarter or period
YEARcharacter(8)Financial year

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
