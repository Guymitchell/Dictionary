User Interface Notes

This table corresponds exactly to the information entered in the Charge Band maintenance program.





Linking And Application

As a standalone table this can be used simply to produce and distribute a master listing for fee earner use.

It could also be used to produce a list of matters where a chargeband with the following link:

MATTR.BAND -> PUB.CHARGEBANDS.BAND
