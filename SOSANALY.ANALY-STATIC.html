Table Name:SOSANALY.ANALY-STATIC

Description: If installed, this view will retrieve data from the SOS data warehouse, returning the configured number of financial periods captured in the warehouse data


Field List:

Field NameData TypeDescription
PREV-BUILD-PERIODSintegerThe number of financial periods of history captured by the data warehouse, configured within the warehouse build parameters file (sosconfig.ini). If the configuration file is changed, the next warehouse build will be able to detect any change in the number of financial periods of data required and so trigger a full rebuild of all data instead of an incremental build.
SEQNOintegerSequence number

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
