Table Name:SOSANALY.ANALY-LOG

Description: If installed, this view will retrieve data from the SOS data warehouse, returning a full detailed warehouse build log


Field List:

Field NameData TypeDescription
DATE-TIMEdatetimeDate-time of log entry
LOG-ENTRYcharacter(200)Warehouse build log details
PROCESS-IDcharacter(40)Process ID of the data warehouse build process
SEQNOintegerLog entry sequence number

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
