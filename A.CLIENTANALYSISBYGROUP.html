View Name:A.CLIENTANALYSISBYGROUP

Description: Provides a summary of time recording and billing information, summarised by client group (for clients allocated to groups) or individual clients


Field List:

Field NameData TypeDescription
CLIENT-OR-GROUPfixcharA hard-coded value of either "CLIENT" or "GROUP" indicating whether the values in each row are for a single client or a consolidation of multiple clients in one particular client group
CLIENT-OR-GROUP-CODEcharacter(255)The client code For a client-level summary row, or the client group code for a group-level summary row
CLIENT-OR-GROUP-NAMEcharacter(475)The client name for a client-level summary row, or the client group name for a group-level summary row
YEARcharacter(254)Year of analysis
TIME-BILLEDdecimalTotal value of time billed in the year (fees billed), equivalent to the X08 nominal balance
TIME-VALUEdecimalTotal value of time recorded in the year
TIME-MINSdecimalTotal number of minutes of time posted in the year
TIME-WOFFdecimalTotal value of time written off in the year
NO-BILLSdecimalTotal number of bills raised in the year
