Table Name:SOSANALY.ANALY-FE

Description: If installed, this view will retrieve data from the SOS data warehouse, returning stub fee earner records derived from the table PUB.FEETR


Field List:

Field NameData TypeDescription
DEPARTMENTcharacter(80)Fee earner department code
FEE-EARNERcharacter(8)Fee earner code
NAMEcharacter(80)Fee earner name
CHARGE-RATEdecimalFee earner's current default charge rate

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
