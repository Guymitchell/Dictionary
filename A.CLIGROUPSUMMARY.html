View Name:A.CLIGROUPSUMMARY

Description: Provides a summary of client group allocation information, with a maximum of one row per client. Clients allocated in mutiple groups are shown as having a client group of "(Multiple)". Clients not in any group are not included.


Field List:

Field NameData TypeDescription
CL-CODEcharacter(254)Client code
FULL-DERIVED-NAMEcharacter(474)Client name (taken from PUB.CLIDB.FULL-DERIVED-NAME)
NUMGROUPSint64Number of client groups to which the client belongs, which is always 1 or more as clients not in any group are excluded
GROUPCODEcharacter(29)Either the group code to which the client is attached, or a dummy code of the form "(x groups)" if the client is in multiple groups
GROUPNAMEfixcharEither the group name of the single client group definition to which the client is attached, or "(Multiple)" if the client is in multiple groups
GROUPDESCRIPTIONfixcharEither the group description of the single client group definition to which the client is attached, or "(Multiple groups)" if the client is in multiple groups
