Practice Manager Interface

This table holds the marketing interests, mailing categories and marketing fee earners associated with a particular organisation or contact. These categories are assigned to contacts and organisations in the "Marketing" tab as shown below.




Field Notes

The MAILINTDATA table holds information related to three other tables: MAILCATEGORIES, MARKETINTERESTS and FEETR (Fee Earner – standard PM table).

Field NameDescription
TYPEWill be set to either:
  • "CO" for contacts
  • "OR" for organisations
REFERENCEHolds either the contact or organisation reference code, validated against either:
  • CONTACTS.REFERENCE-CHAR
  • ORGANISATION.REFERENCE-CHAR
CATContains a single code letter with one of three possible values:
  • "I" for market interest (MARKETINTERESTS)
  • "F" for fee farners (FEETR)
  • "M" for mailing category (MAILCATEGORIES)
CAT-CODEDepending upon the code letter in CAT above, CAT-CODE will match either:
  • When CAT = "I", CAT-CODE = MARKETINTERESTS.CODE
  • When CAT = "F", CAT-CODE = FEETR.FEE-EARNER
  • When CAT = "M", CAT-CODE = MAILCATEGORIES.CODE

In this screen, when a Profile (MARKETPROFILES) is highlighted, the table MARKET-PRO-MAIL checks to see if a "group" of Mailing Categories exists (the linking field is min-pro-code between MARKETPROFILES and MARKET-PRO-MAIL). While the grouping function has a wide range of uses, its specific use on this screen is to highlight which mailing categories are part of the group in order to assist the user in selecting all the grouped items in one selection operation.
