Table Name:SOSANALY.ANALY-FEES-BILLED

Description: If installed, this view will retrieve data from the SOS data warehouse, returning fees billed and original time value summary information


Field List:

Field NameData TypeDescription
BRANCHcharacter(8)Branch code
FEE-EARNERcharacter(8)Fee earner that fees billed value relates to
MT-TYPEcharacter(8)Matter type code
MT-CODEcharacter(60)Matter code
CL-CODEcharacter(30)Client code
SUPERVISORcharacter(8)Matter supervisor code
YEARcharacter(8)Financial year
PERIODintegerFinancial period
FEES-BILLEDdecimalFees billed value for this fee earner in this year and period
ORIG-TIME-VALUEdecimalOriginal value of time included in bills for the fee earner in this year and period

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
