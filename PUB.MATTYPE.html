Practice Manager Interface

This table holds matter type maintenance information as defined in Practice Manager matter type maintenance screen.




Field Notes

DESTRUCTION-DAYS has a default value of 2200, i.e. the equivalent of 6 years rounded up.

CAT-CODE can be used to look up the category description in the table LACAT. DEPARTMENT can be used to look up the department name in the table DEPARTMENT. These joins should be Left Outer joins because they may be blank in your MATTYPE table, especially if data has recently been converted from another non-SOS Practice Manager database.

You should note that any information entered in the Archive Additional Tab Handling panel of the Matter Type maintenance screen is stored not in MATTYPE but in the TYPEREC table. Each of the five fields if used will generate a distinct TYPEREC record with values as follows:

Linking And Application

Simple example to provide a matter listing grouped by matter types and showing their descriptions:

MATDB.MT-TYPE -> MATTYPE.MT-TYPE


Two joins to provide a matter listing grouped by matter type within department:

MATDB.MT-TYPE -> MATTYPE.MT-TYPE

MATTYPE.DEPARTMENT -> DEPARTMENT.DEPARTMENT-CODE
