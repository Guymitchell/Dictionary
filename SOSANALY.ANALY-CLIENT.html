Table Name:SOSANALY.ANALY-CLIENT

Description: If installed, this view will retrieve data from the SOS data warehouse, returning stub client records derived from the table PUB.CLIDB


Field List:

Field NameData TypeDescription
CL-CODEcharacter(30)Client code (taken from CLIDB.CL-CODE)
NAMEcharacter(200)Full client name (taken from CLIDB.FULL-DERIVED-NAME)
FEE-EARNERcharacter(8)Client fee earner/supervising partner code (taken from CLIDB.FEE-EARNER)

NOTE that all data in the SOS warehouse is built periodically into the warehouse from the main SOS database, and as such the warehouse contains a snapshot of live data at the point the warehouse build was last run. The warehouse does not show changes real-time as they posted into the main live SOS database.
