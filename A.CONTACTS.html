This table corresponds almost entirely to the fields available for input in the Individual Create and Amend programs accessed from the General Search screen.



Field Notes

REFERENCE-CHAR holds the same numbers as REFERENCE but held as a character value instead of a numeric value. This allows you to link CONTACTS to other tables such as MATTRELATIONS where the matching code is held as a string, as Crystal will not always report reliably if a string to number link is attempted.



Suggested Linking

The CONTACTS table links to many other tables, for example:

To list additional telephone numbers for a contact:

PUB.TELEPHONES.REFERENCE -> PUB.CONTACTS.REFERENCE-CHAR (where PUB.TELEPHONES.TYPE = "CO")


To list contacts who are parties of clients:

A.CONTACTS.CON-REFERENCE -> PUB.CONTRELATIONS.CON-REFERENCE

A.CONTRELATIONS.REL-RECORD -> PUB.CLIDB.CL-CODE

Where PUB.CONTRELATIONS.ROL-CODE = "C* and PUB.CONTRELATIONS.REL-FILE = "CL"
